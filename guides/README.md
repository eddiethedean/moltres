## Quick Navigation

**New to Moltres?**
1. Start with [Getting Started](https://moltres.readthedocs.io/en/latest/guides/getting-started.html)
2. Review [Common Patterns](https://moltres.readthedocs.io/en/latest/guides/common-patterns.html) for real-world examples
3. Read [Best Practices](https://moltres.readthedocs.io/en/latest/guides/best-practices.html) as you write code

**Coming from Pandas?**
1. Read [Migrating from Pandas](https://moltres.readthedocs.io/en/latest/guides/migrating-from-pandas.html)
2. Learn the [Pandas Interface](https://moltres.readthedocs.io/en/latest/guides/pandas-interface.html) for pandas-style operations
3. Check [Common Patterns](https://moltres.readthedocs.io/en/latest/guides/common-patterns.html) for equivalent operations
4. Review [Performance Optimization](https://moltres.readthedocs.io/en/latest/guides/performance-optimization.html) for efficiency tips

**Coming from PySpark?**
1. Read [Migrating from PySpark](https://moltres.readthedocs.io/en/latest/guides/migrating-from-pyspark.html)
2. Most PySpark code works with minimal changes!
3. See [Advanced Topics](https://moltres.readthedocs.io/en/latest/guides/advanced-topics.html) for advanced features

**Having Issues?**
1. Check [Error Handling](https://moltres.readthedocs.io/en/latest/guides/error-handling.html) for debugging techniques
2. Review [Troubleshooting](https://moltres.readthedocs.io/en/latest/TROUBLESHOOTING.html) in the main docs
3. See [FAQ](https://moltres.readthedocs.io/en/latest/FAQ.html) for common questions
