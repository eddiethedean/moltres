# dbt Example Project with Moltres

This is an example dbt project demonstrating how to use Moltres DataFrames in dbt Python models.

## Setup

1. Install dependencies:
```bash
pip install moltres[dbt] dbt-core
```

2. Configure your dbt profile (see dbt documentation)

3. Run models:
```bash
dbt run
```

## Models

- `example_model.py` - Example Python model using Moltres

See the main dbt integration guide for more details.
